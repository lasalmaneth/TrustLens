# TrustLens
TrustLens is an intelligent, human-centered cybersecurity assistant built to counter social engineering attacks in SMS, email, and chat messages.

Instead of relying only on keyword filters or URL checks, TrustLens uses a collaborative multi-agent AI architecture to detect psychological triggers such as artificial urgency, fear, authority pressure, and financial temptation.

The system provides:
- A comprehensive risk score
- An explainable breakdown of detected manipulation tactics
- Actionable safety recommendations to help prevent digital scam victimization

Planned platform stack: FastAPI, Next.js, and advanced LLM orchestration.
